Git -  це розподілена система керування версіїями файлів
В якому статусі можуть перебувати файли Git - Tracked або Untracked
Для чого потрібна команда add -  Для того, щоб файли відстежувалися
Для чого потрібна команда status - Для перевірки, чи відстежуються файли чи ні
Для чого потрібна команда commit - Для збереження версії файлів на момент виконання команди add
Для чого потрібна команда init -  Для ініціалізації локального репозиторію
В коментарях комітів потрібно писати усі зміни, що було виконані або додані
Щоб переглянути історію комітів, потрібно використати команду git log
Для чого потріна команда checkout - команда для переміщення між гілками
